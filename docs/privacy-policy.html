Flight Detector - Privacy Policy

Last updated: December 2024

Overview

Flight Detector is a Chrome extension that identifies aircraft makes (Boeing, Airbus, others) on flight booking websites. This privacy policy explains how we handle your data.

Data Collection

We do not collect, store, or transmit any personal information.

  • No user accounts or profiles are created
  • No personal information is collected
  • No browsing history is stored
  • No flight booking data is transmitted to our servers

Data Processing

The extension processes flight information locally on your device:

  • Flight Numbers: Extracted from web pages to identify aircraft types
  • Aircraft Data: Retrieved from public APIs to display aircraft information
  • Missing Flights: Stored locally in Chrome storage for tracking purposes

External APIs

The extension uses the following external services:

  • Vercel API: Public aircraft database API to fetch aircraft information
  • Cleartrip API: Public flight search API (when available) to get aircraft data

These APIs only receive flight numbers (e.g., "IC5302") and return aircraft type information. No personal data is shared.

Local Storage

The extension stores the following data locally in your Chrome browser:

  • Missing flight numbers (for tracking purposes)
  • Aircraft information cache (to reduce API calls)

This data is stored locally and never transmitted to external servers.

Permissions

The extension requests the following permissions:

  • Active Tab: To read flight information from web pages
  • Storage: To store missing flight data locally
  • Host Permissions: To access Cleartrip.com and the Vercel API

Data Security

We implement the following security measures:

  • All data processing happens locally on your device
  • No personal information is transmitted to external servers
  • API calls only include flight numbers, not personal data
  • Local storage is encrypted by Chrome's built-in security

Third-Party Services

The extension integrates with:

  • Chrome Extension API: For browser integration
  • Vercel API: For aircraft database access
  • Cleartrip API: For flight information (when available)

These services have their own privacy policies, which we recommend reviewing.

Data Retention

Data is retained as follows:

  • Missing Flights: Stored locally until manually cleared
  • Aircraft Cache: Stored locally and automatically managed
  • No Server Storage: We do not store any data on our servers

Your Rights

You have the following rights regarding your data:

  • Clear Data: Use the "Clear Database" button in the extension popup
  • Uninstall: Remove the extension to delete all local data
  • Export Data: Download missing flight data using the popup

Children's Privacy

This extension does not knowingly collect personal information from children under 13. The extension is designed for general audiences and does not require user registration.

Changes to Privacy Policy

We may update this privacy policy from time to time. Users will be notified of any material changes through the Chrome Web Store update process.

Contact Information

If you have any questions about this privacy policy or the extension's data practices, please contact us through the Chrome Web Store developer contact information.

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR) principles
  • California Consumer Privacy Act (CCPA) requirements
